Leafly Integration Architecture

How do you envision the data flowing into Verdant Apothecary?

A

Official API (Real-time)

Direct integration with Leafly's B2B API to pull strain data dynamically.

B

Automated Scraper

A background worker (e.g. Puppeteer/Cheerio) that crawls Leafly periodically to build our internal database.

C

Batch Import (Static)

Admin uploads CSVs or JSON files manually to seed the strain database, mapping columns to our schema.
