Triggering the Synchronization

Since we are syncing and storing the data, how should we trigger the API pulls?

A

Automated Cron Job (Nightly/Weekly)

Set up a background job (e.g. via Supabase Edge Functions + pg_cron or Next.js Cron) to automatically pull new strains and updates on a regular schedule.

B

Manual Admin Sync

Build an internal Admin Dashboard with a "Sync from Leafly" button, so clinical staff can control exactly when the database gets updated.

C

Both (Automated + Manual Override)

Have a nightly automated job, but also provide an admin UI to manually trigger a sync for immediate updates or debugging.
