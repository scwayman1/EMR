Data Synchronization Strategy

How should we store the API data within Verdant Apothecary?

A

Sync & Store (Recommended)

We fetch from Leafly API and save them as native `ChemovarRecord`s in our Postgres DB. Gives us full control, fast local searches, and lets us add custom tags.

B

Fetch On-Demand

We query Leafly in real-time when a user searches. No need to manage DB storage or syncing, but relies 100% on their API uptime and limits.

C

Hybrid / Caching

Fetch on demand, but cache the result in our DB for 30 days. Balances storage with performance.
