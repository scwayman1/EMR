EMR-028: Split Window / Multi-Tab Workspace

Designed by Dr. Neal Patel — select the workspace split model that would offer the best clinical flow.

A

Dynamic Split Panes (Bloomberg Style)

A flexible grid layout supporting up to 3 vertical or horizontal split panes.

Pros:
  • Simultaneous views of multiple patient profiles, notes, or prescriptions.
  • Resizable boundaries by dragging grid splitters.
Cons:
  • Can feel cluttered on smaller screens (e.g. iPad portrait).
  • Higher rendering overhead.
B

Tabbed Workspace with Docking (VS Code Style)

A tab bar at the top of the workspace. Tabs can be dragged and docked side-by-side or split.

Pros:
  • Highly organized and familiar tabs for different patient visits or research library.
  • Supports arbitrary number of open tabs; up to 3 active splits.
Cons:
  • Managing too many tabs might require explicit closure flows.
C

Primary Workspace + Dual Sliders (Hybrid Style)

A centered main workspace with collapsible, pinnable left/right side panels for quick lookup.

Pros:
  • Maintains focus on the primary action (e.g. typing notes or prescribing).
  • Quick peek at vitals, history, or dosage guidelines without context switching.
Cons:
  • Less flexible if you need 3 fully interactive custom views side-by-side.
